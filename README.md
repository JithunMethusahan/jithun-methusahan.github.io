# Portfolio

This repository contains my personal portfolio website with a GitHub-backed blog.

## How the blog works

- Add a new markdown file inside the blog-content folder.
- The site will load it automatically when published on GitHub Pages.
- Use frontmatter like this at the top of the file:

```md
---
title: My Post Title
published: 2026-07-09
estimated read time: 8 minutes
---

Your content here.
```

## GitHub Pages deployment

1. Push this repository to GitHub.
2. Open the repository settings.
3. Go to Pages.
4. Select Deploy from a branch.
5. Choose the main branch and the root folder.
6. Save and wait for the site to publish.
